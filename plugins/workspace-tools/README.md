# Workspace Tools Plugin

Bundled productivity tools for creating and editing common workspace artifacts.

The plugin owns the authoritative skill content. It is installed from the
marketplace independently of the Cowork platform runtime, which only supplies
the executables and libraries these skills invoke.

## Skills

- `documents`
- `pdf`
- `presentations`
- `spreadsheets`

## Purpose

Use this plugin when a user needs to create, edit, analyze, or verify documents,
PDFs, presentation decks, spreadsheets, or related files in a workspace.
